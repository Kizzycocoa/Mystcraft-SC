# Mystcraft-SC

Mystcraft: The Synthetic Codex is an attempt to port Mystcraft from 1.12.2 to 1.21.11

This project initially was set to be a personal project. As a result of some bad-faith actions taken against me, I am deciding it shall become a more public release.
